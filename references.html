Dataset Manipulation

The dataset chosen for this data visualisation was discussed and chosen with all team members present. Changes were made to the dataset, however said changes did not result in any major deviation from the overarching topic at hand. The changes to the dataset were as follows:

  • Removed data related to irrelevant illnesses (which created entirely different circumstances).
  • Removed countries that contained missing dataset within the chosen time period.
  • Split into two separate sections for audiences to compare and contrast.
